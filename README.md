## Panduan Penggunaan

### 1. Instalasi
1. Copy semua file ke dalam folder web server (htdocs atau www)
2. Import database dari file vehicle_booking.sql ke MySQL
3. Configure database connection di application/config/database.php
4. Pastikan folder application/cache memiliki izin write

### 2. Login
1. Akses aplikasi melalui browser: http://localhost/[nama_folder]
2. Masukkan username dan password sesuai dengan role
